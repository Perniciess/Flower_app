● Вот что уже реализовано и чего не хватает:                                                                                                                                             
                                                                                                                                                                                         
  Реализовано                                                                                                                                                                            
                                                                                                                                                                                         
  - Auth — регистрация через Telegram, логин, JWT, сброс пароля, WebSocket                                                                                                               
  - Users — CRUD, RBAC (client/admin)                                                                                                                                                    
  - Flowers — каталог с изображениями, CRUD для админа                                                                                                                                   
  - Carts — корзина с товарами, добавление, изменение количества                                                                                                                         
  - Orders — создание заказа, оплата через YooKassa, вебхук                                                                                                                              
  - Security — CSRF, Argon2, токен-блеклист в Redis                                                                                                                                      
                                                                                                                                                                                         
  Что не хватает                                                                                                                                                                         
                                                                                                                                                                                         
  Критичное:                                                                                                                                                                             
  1. Получение заказов — нет эндпоинтов для просмотра истории заказов пользователем и списка заказов для админа                                                                          
  2. Удаление товара из корзины — есть добавление и изменение, но нет DELETE /carts/cart_item/{id}                                                                                       
  3. Управление статусами заказов — статусы IN_PROGRESS, ON_THE_WAY, DELIVERED объявлены в enum, но нет эндпоинтов для их изменения админом                                              
  4. Управление наличием — у Flower нет поля stock/количества, нет проверки доступности при заказе                                                                                       
                                                                                                                                                                                         
  Важное:                                                                                                                                                                                
  5. Поиск и фильтрация — нет фильтрации цветов по цене, цвету; нет пагинации                                                                                                            
  6. Адрес доставки — нет модели адреса, нет привязки к заказу                                                                                                                           
  7. Категории — у цветов нет категорий (букеты, розы, тюльпаны и т.д.)                                                                                                                  
                                                                                                                                                                                         
  Желательное:                                                                                                                                                                           
  8. Уведомления — нет оповещений о смене статуса заказа (Telegram/email)                                                                                                                
  9. Промокоды/скидки — нет системы купонов                                                                                                                                              
  10. Избранное — нет вишлиста                                                                                                                                                           
  11. Отзывы — нет рейтингов и отзывов на товары                                                                                                                                         
                                                                                                                                                                                         
  Пункты 1–4 нужны для минимально рабочего магазина. Хочешь, чтобы я начал с какого-то из них?
